## 多利亚小镇

### 简介
这个项目使用javascript重写了[像素地牢](https://github.com/watabou/pixel-dungeon/)的一小部分（和它的一个改版：[特别惊喜像素地牢](https://github.com/hmdzl001/SPS-PD/)中的一张地图）。你可以把它叫做“有着像素地牢风格的网页”。
### 更新
相较于上一个版本，这个经历了重构过后的版本提升了不少性能，并增加了一些内容。
